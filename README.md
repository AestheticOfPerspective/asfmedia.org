# asfmedia.org

Static landing page for ASF Media.

## Positioning

ASF Media connects photography, video, web presence, and digital systems into a lighter, more practical client-facing surface.

This repo is intentionally simple:

- plain static HTML
- no framework
- no build step
- easy to deploy from a git repo

## Current Goal

Use this repo as a clean, deployable base for the ASF Media web relaunch, with `web.asfmedia.org` as the current public reference surface.

## Current Workflow Direction

The intended flow is:

- local edits in this repo
- push to GitLab as the source of truth
- connect the GitLab repo to the chosen static deploy target
- change DNS only after billing, hosting, and preview validation are confirmed
